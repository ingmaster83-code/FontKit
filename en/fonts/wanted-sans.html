About this Font

Wanted Sans is an open-source Korean font created by Wanted Lab, the recruitment platform. Optimized for IT startup branding with a modern, refined design, it is a variable font covering Thin (100) through ExtraBold (800) — and is actually used in the Wanted product itself.

Recommended Use Cases

  • IT startup branding
  • Recruitment & HR UI
  • Tech product design
  • App & web interfaces
  • B2B SaaS

Web Font CSS

@import url('https://cdn.jsdelivr.net/gh/wanteddev/wanted-sans@v1.0.3/packages/wanted-sans/fonts/static/woff2/WantedSans.css');
body { font-family: 'Wanted Sans', sans-serif; }

Tips

Frequently Asked Questions

Can I use Wanted Sans commercially?

Yes, under OFL — free for personal and commercial use.

How does it compare to Pretendard?

Both are modern gothic, but Wanted Sans leans sharper and more technical, inspired by IT services.

Are files uploaded to the internet?

No. This page only provides font information and a download link. No files are transmitted to any server.
