About this Font

Gyeonggi Millennium is an official typeface created and distributed by Gyeonggi Province to commemorate its millennium. It comes in two styles — Title and Text — with a clean, modern design suitable for both public and private design projects.

Recommended Use Cases

  • Public institutions & local government
  • Report & document body text
  • Web & app UI
  • Brochures & pamphlets

Web Font CSS

/* 경기도 공식 배포처에서 다운로드 후 사용 */
@font-face {
  font-family: 'GgTitle';
  src: url('GgTitle.woff2') format('woff2');
}
body { font-family: 'GgTitle', sans-serif; }

💡 Tips

  • Commercial use is free under the Gyeonggi Province public license.
  • The Title style is optimized for headlines; the Text style for long-form reading.
  • Using both styles together creates a cohesive typographic hierarchy.

Frequently Asked Questions

Can I use Gyeonggi Millennium commercially?

Yes, the Gyeonggi Province public license permits free personal and commercial use.

What is the difference between Title and Text styles?

Title is optimized for headlines and emphasis; Text is designed for comfortable long-form reading.

Are files uploaded to the internet?

No. This page only provides font information and a download link.
