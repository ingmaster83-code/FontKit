About this Font

Source Serif 4 is an open-source serif family from Adobe. Optimized for long-form screen reading, it suits content requiring trustworthiness — news, media, documents. Optical-size variable font supporting ExtraLight (200) through Black (900).

Recommended Use Cases

  • News & media sites
  • Long-form articles & blogs
  • Academic documents
  • Corporate reports
  • Digital publishing

Web Font CSS

@import url('https://fonts.googleapis.com/css2?family=Source+Serif+4:opsz,wght@8..60,200..900&display=swap');
body { font-family: 'Source Serif 4', serif; }

Tips

  • License: OFL. Free for commercial use.
  • Pair with Adobe's Source Sans 3 for a unified sans-serif and serif system.
  • Optical-size variable font — excellent readability even at small sizes.

Frequently Asked Questions

Can I use Source Serif 4 commercially?

Yes, under OFL — free for personal and commercial use.

How does it differ from Source Serif Pro?

Source Serif 4 is an updated version of Source Serif Pro, adding variable font support and optical size adjustment.

Are files uploaded to the internet?

No. This page only provides font information and a download link. No files are transmitted to any server.
