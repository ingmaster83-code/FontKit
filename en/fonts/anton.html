About this Font

Anton by Vernon Adams is a bold, intense display sans-serif. One of the most widely used display fonts worldwide in YouTube thumbnails, social media accent text, and streaming content. One weight only, but the go-to choice for immediate impact.

Recommended Use Cases

  • YouTube thumbnail text
  • Social media accent copy
  • Streaming content titles
  • Event banners
  • App onboarding emphasis

Web Font CSS

@import url('https://fonts.googleapis.com/css2?family=Anton&display=swap');
body { font-family: 'Anton', sans-serif; }

Tips

  • License: OFL. Free for commercial use.

Frequently Asked Questions

Can I use Anton commercially?

Yes, under OFL — free for personal and commercial use.

How is it different from Bebas Neue?

Both are bold all-caps display fonts, but Bebas Neue is taller and narrower while Anton is wider and slightly rounder.

Are files uploaded to the internet?

No. This page only provides font information and a download link. No files are transmitted to any server.
