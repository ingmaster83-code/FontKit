폰트 소개

배달의민족 도현체는 두껍고 강렬한 느낌의 한글 디스플레이 폰트입니다. 포스터, 배너, 광고 헤드라인에서 강렬한 인상을 남깁니다. 우아한형제들이 무료로 배포합니다.

추천 활용 분야

  • 포스터·배너 헤드라인
  • 광고·홍보물
  • 유튜브 썸네일
  • 브랜드 슬로건

웹폰트 CSS 사용법

/* 공식 배포처에서 다운로드 후 사용 */
@font-face {
  font-family: 'BMDohyeon';
  src: url('BMDohyeon.woff2') format('woff2');
}
body { font-family: 'BMDohyeon', sans-serif; }

💡 Tips

  • 라이선스는 OFL로, 상업 이용 시 별도 허가가 필요 없습니다.
  • Regular 단일 굵기로 제공되며, 임팩트 있는 헤드라인 전용으로 활용하세요.
  • Black Han Sans와 함께 비교해 보면 스타일 차이를 확인할 수 있습니다.

자주 묻는 질문

배달의민족 도현체를 상업적으로 사용해도 되나요?

네, OFL 라이선스 하에 개인·상업 목적 모두 무료 사용 가능합니다.

굵기가 하나만 있나요?

도현체는 Regular 단일 굵기로 제공됩니다. 다양한 굵기가 필요하다면 한나체 Pro나 에스코어 드림을 검토해 보세요.

파일이 인터넷에 업로드되나요?

아니요. 이 페이지는 폰트 정보와 다운로드 링크만 제공합니다.
