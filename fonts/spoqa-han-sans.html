폰트 소개

스포카 한 산스는 스포카에서 제작한 한글·일본어·영문 혼용 폰트입니다. Adobe와 구글이 공동 제작한 Source Han Sans를 기반으로, 영문 부분을 Noto Sans로 교체하여 한·일·영 혼용 환경에서 더 자연스러운 타이포그래피를 제공합니다.

추천 활용 분야

  • 한·일·영 혼용 문서
  • 세련된 브랜드 디자인
  • 앱 인터페이스
  • 출판·인쇄물

웹폰트 CSS 사용법

@import url('https://spoqa.github.io/spoqa-han-sans/css/SpoqaHanSansNeo.css');
body { font-family: 'Spoqa Han Sans Neo', sans-serif; }

💡 Tips

  • 스포카 한 산스 라이선스는 OFL (오픈 폰트 라이선스)입니다. 상업적 이용 시 폰트 파일 재판매만 금지됩니다.
  • 웹폰트로 사용 시 아래 CSS 코드를 그대로 복사해 <head> 태그 안에 붙여넣으세요.
  • 폰트 미리보기 텍스트를 수정하려면 아래 텍스트 박스를 클릭해 직접 입력해 보세요.

자주 묻는 질문

스포카 한 산스을(를) 상업적으로 사용해도 되나요?

OFL (오픈 폰트 라이선스) 하에 제공되므로 개인·상업 목적 모두 무료로 사용할 수 있습니다. 폰트 파일 자체를 유료로 재판매하는 것은 허용되지 않습니다.

인쇄물이나 영상에도 사용할 수 있나요?

네, 웹·앱뿐만 아니라 인쇄물, 광고, 영상 자막, SNS 콘텐츠 등 다양한 매체에 자유롭게 사용할 수 있습니다.

파일이 인터넷에 업로드되나요?

아니요. 이 페이지는 폰트 정보와 다운로드 링크만 제공합니다. 파일을 서버로 전송하는 기능은 없습니다.
