폰트 소개

에스코어 드림은 S-Core가 제작·배포하는 고품질 한글 고딕 폰트입니다. 굵기 1(Thin)부터 9(Heavy)까지 총 9단계 굵기를 지원하여 본문부터 디스플레이까지 폭넓게 활용할 수 있습니다. 국문·영문·숫자·특수문자를 모두 포함하며, OFL 라이선스로 상업 이용이 가능합니다.

추천 활용 분야

  • 브랜딩 · UI 디자인
  • 프레젠테이션 슬라이드
  • 포스터 · 배너
  • 웹사이트 본문

웹폰트 CSS 사용법

@font-face {
  font-family: 'S-CoreDream';
  src: url('https://cdn.jsdelivr.net/gh/projectnoonnu/noonfonts_six@1.2/S-CoreDream-4Regular.woff') format('woff');
  font-weight: 400;
}
body { font-family: 'S-CoreDream', sans-serif; }

💡 Tips

  • 굵기 4(Regular)와 7(Bold)을 조합하면 강약 있는 타이포그래피를 만들 수 있습니다.
  • 얇은 굵기(1~3)는 럭셔리·미니멀 디자인에 특히 잘 어울립니다.
  • OFL 라이선스이므로 상업 이용 시 폰트 재판매만 금지됩니다.

자주 묻는 질문

에스코어 드림을 상업적으로 사용해도 되나요?

OFL (오픈 폰트 라이선스)로 배포되어 개인·상업 목적 모두 무료로 사용할 수 있습니다.

굵기별로 파일이 따로 있나요?

네, 굵기 1~9 각각 별도의 폰트 파일로 제공됩니다. 필요한 굵기만 선택해 다운로드할 수 있습니다.

파일이 인터넷에 업로드되나요?

아니요. 이 페이지는 폰트 정보와 다운로드 링크만 제공합니다.
