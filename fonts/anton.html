폰트 소개

Anton은 Vernon Adams가 제작한 굵고 강렬한 디스플레이 산세리프 폰트입니다. 유튜브 썸네일, 소셜 미디어 강조 텍스트, 스트리밍 콘텐츠에서 전 세계적으로 가장 많이 쓰이는 디스플레이 폰트 중 하나입니다. 단일 굵기이지만 즉각적인 임팩트를 원할 때 최적의 선택입니다.

추천 활용 분야

  • 유튜브 썸네일 텍스트
  • 소셜 미디어 강조 문구
  • 스트리밍 콘텐츠 타이틀
  • 이벤트 배너
  • 앱 온보딩 강조

웹폰트 CSS 사용법

@import url('https://fonts.googleapis.com/css2?family=Anton&display=swap');
body { font-family: 'Anton', sans-serif; }

Tips

  • 라이선스는 OFL입니다. 상업적 이용 가능합니다.
  • Regular 한 가지 굵기만 있으며 영문 전용입니다.
  • 흰색 텍스트에 짙은 그림자(text-shadow)를 추가하면 유튜브 썸네일 느낌이 완성됩니다.

자주 묻는 질문

Anton을 상업적으로 사용해도 되나요?

OFL 라이선스로 배포되어 개인·상업 목적 모두 무료입니다.

Bebas Neue와 어떻게 다른가요?

두 폰트 모두 굵은 올캡스 디스플레이 폰트지만, Bebas Neue는 더 세로로 길고 좁은 반면 Anton은 좀 더 넓고 둥근 편입니다.

파일이 인터넷에 업로드되나요?

아니요. 이 페이지는 폰트 정보와 다운로드 링크만 제공합니다. 파일을 서버로 전송하는 기능은 없습니다.
