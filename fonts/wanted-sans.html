폰트 소개

Wanted Sans는 채용 플랫폼 원티드랩이 자체 제작해 오픈소스로 공개한 한글 폰트입니다. IT 스타트업 브랜딩에 최적화된 현대적이고 세련된 디자인이 특징이며, Thin(100)부터 ExtraBold(800)까지 가변폰트로 제공됩니다. 실제 원티드 서비스에서 사용 중인 폰트이기도 합니다.

추천 활용 분야

  • IT 스타트업 브랜딩
  • 채용·HR 서비스 UI
  • 테크 프로덕트 디자인
  • 앱·웹 인터페이스
  • B2B SaaS

웹폰트 CSS 사용법

@import url('https://cdn.jsdelivr.net/gh/wanteddev/wanted-sans@v1.0.3/packages/wanted-sans/fonts/static/woff2/WantedSans.css');
body { font-family: 'Wanted Sans', sans-serif; }

Tips

  • 라이선스는 OFL입니다. 상업적 이용 가능합니다.
  • 가변폰트(Variable Font)라서 100~800 모든 굵기를 하나의 파일로 사용할 수 있습니다.
  • 실제 원티드 서비스 UI에 쓰이는 폰트라 IT 서비스 느낌이 자연스럽게 납니다.

자주 묻는 질문

Wanted Sans를 상업적으로 사용해도 되나요?

OFL 라이선스로 배포되어 개인·상업 목적 모두 무료입니다.

Pretendard와 비교하면 어떤가요?

비슷한 현대적 고딕 계열이지만 Wanted Sans는 IT 서비스에서 영감을 받아 좀 더 날카롭고 기술적인 느낌을 줍니다.

파일이 인터넷에 업로드되나요?

아니요. 이 페이지는 폰트 정보와 다운로드 링크만 제공합니다. 파일을 서버로 전송하는 기능은 없습니다.
